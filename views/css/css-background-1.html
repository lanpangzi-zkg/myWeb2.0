css背景属性探讨一

发布时间：2017-02-28 标签:css background-clip&background-origin

在工作场景当中，时常用到background属性，使用最多当属background-color,background-image这些了，但最近看到了《css secrets》这本书后，对背景有了新的认识和了解，今天就总结一下background-clip,background-origin和background-position这三个属性。在解释这几个属性前，需要了解一下盒模型(下面这个图片从网上盗的，自己懒得画了~):

1.background-clip

这个属性从字面上讲就是裁剪，当然它也就是干这个的，它有三个取值：

border-box | padding-box | content-box

  • 1、border-box:此值为默认值，背景从border区域向外裁剪，也就是超出部分将被裁剪掉；
  • 2、padding-box：背景从padding区域向外裁剪，超过padding区域的背景将被裁剪掉；
  • 3、content-box：背景从content区域向外裁剪，超过context区域的背景将被裁剪掉；

简单了解了一下属性之后，接下来就看一下实例：

----html----
<div class="box>
哈哈哈哈......
</div>
----css----
.box{
      width: 10em;
      height: 10em;
      background-color: #fbde7c;
      margin: 1em auto;
      border:1em dotted #eee;
      padding:1em;
      color: #fff;
      background-repeat: no-repeat;
 }

默认情况下背景颜色会延伸到边框下面，这里面框特意处理成dotted类型，通过效果图可以明显看到背景在边框下面，而通过设置backgound-clip就可以裁剪背景;

上面的实例当中，background-clip默认是border-box，现在我们来设置它，感受一下不同的效果：

background-clip:padding-content

background-clip:content-box

通过对比就可以知道background-clip的用处了，这里还需要强调一点，该属性是css3新增的，对background-color和background-image均有效。

2.background-origin

background-origin表示背景图片开始绘制的位置，也就是起点，和background-clip取值一样，但默认值是padding-box,接下来的实例我们在前面的例子当中稍作修改，添加一张背景图片，同时去掉background-color和background-clip;

background-origin:padding-box(默认情况下)

background-origin:border-box

background-origin:content-box

通过对比可以发现，不管哪一种背景的起点，终点都是border-box的右下角；

3.background-position

background-position表示背景图片相对于background-origin的偏移位置，这里需要注意的是，使用该属性之前需要设置背景图片，否则无效，它的取值方式多样，可以使left|top|right|bottom，px或者%，接下来还是以上面的实例为基础：

上面两幅图中，左边为默认情况，右边设置了background-position:10px 10px,可以明显看到图片相对于起点有明显的偏移；

上面两幅图是设置了background-origin:border-box的对比效果；
