python基础学习--面向对象

发布时间：2016-12-14 标签:python

python作为一门高级开发语言，面向对象是其不可或缺的重要特性，何为面向对象，万物皆对象，高度抽象，而又映射到具体实体上，python中的概念和语法跟java很类似，有java基础的看到python这块内容会有很熟悉的感觉，下面将记录一下面向对象在python中的实现。

1.定义

class Person:
   pass

备注：

1.class关键字定义类，可省略
2.默认继承object超类
3.类名首字母大写

2.创建实例

p1=Person()
p1.name='neo'
print(p1.name) # neo

备注：

1.创建实例没有new关键字
2.可以动态给实例添加属性和方法

3.构造函数

说明：与java类似，python中也有构造函数的概念，用来初始化实例，在具体实现上面，略有差异 

class Person:
  def __init__(self,name,age):
       self.name=name
       self.age=age
p1=Person('neo',20)
print(p1.name,p1.age) # neo 20
备注：

self表示当前实例，与Java中的this关键字类似

4.普通方法

class Person:
  def __init__(self,name,age):
       self.name=name
       self.age=age
  def say(self):
     print(self.name,self.age)
  def sayHello(self,words):
     print(self.name,words)
p1=Person('neo',20)
p1.say() # neo 20
p1.sayHello('hello') # neo hello

备注：

1.普通函数第一个参数也是self
2.self后面添加入参

5.属性访问限制

说明：如果想隐藏某些重要数据，限制外界的直接访问，这时候就需要限制属性的访问，提供统一对外接口，封装内部数据逻辑，功能于类似java中的private关键字。 

def Person: 
   def __init__(self,name,age)
       self.__name=name
       self.age=age
   def getName(self):
       return self.__name

            
p1=Person('neo',20)

            
print(p1.age) # 20
    
            
print(p1.getName()) # 'neo' 
 
            
p1.__name # 'Person' object has no Attribute '__name'

  
            

              备注：
            

            

             __xx,双下划线+属性名，这类属性不能通过实例直接访问，需要暴露接口调用

            


            
6.类属性和实例属性

            

              说明：类属性从属于类，所有实例共享，实例属性从属于某一个具体的实例，属于独有；
            

            
class Person:

            
   count=0

            
   def __init__(self.name):

            
     self.name=name

            
     self.age=age

            
p1=Person('neo',20)

            
print(p1.count) # 0

            
print(Person.count) # 0

            
p1.count=2

            
p2=Person('jack',30)

            
print(p2.count) # 0

            


            

              备注：
            

            

             1.p1和p2共享类属性count

             2.类属性可以通过类名访问

             3.在实例上试图修改类属性是不能成功的，其实是在实例中添加了同名的属性，在访问变量的时候，有类似js中作用域的概念，先从实例中查找，如果没有再从类中查找

            


            
6.类方法和静态方法

            

              说明：类方法使用@classmethod标注，静态方法使用@staticmethod标注；
            

            
class Person:

            
   num=10

            
   def __init__(self.name):

            
       self.name=name

            
   @classmethod

            
   def classm(cls):

            
       print(cls.num)

            
   @staticmethod

            
   def staticm(name):

            
       print(name)


            
p1=Person('neo')

            
print(p1.num) # 10

            
print(Person.num) # 10

            
Person.staticm('neo') # neo


            
p2=Person('jack')

            
p2.classm() # 10

            
Person.staticm('jack') #jack

             

              备注：
            

            

              1.实例方法第一个参数是self

              2.类方法第一个参数是class,类方法只能访问类变量

              3.类方法可以通过类名或者实例调用

              4.静态方法不需要额外的参数，不能访问类变量和实例变量

              5.静态方法通过类名调用

            


            
7.继承

            

              说明：python可以多继承，这与java有点区别，java是能单继承；
            

            
class Person:

            
   def __init__(self,name):

            
     self.name=name

            
   def sayName(self):

            
       print(self.name)


            
class Student(Person):

            
   def __init__(self,name,school):

            
       super(Student,self).__init__(name)

            
       self.school=school

            
   def sayName(self):

            
       print('this is student',self.name)


            
p1=Person('neo')

            
p1.sayName() # neo

            
s1=Student('xiaoming','ccnu') # this is student xiaoming

            
s1.sayName()

             

              备注：
            

            

             1.如果一个子类继承多个基类，那么对于基类存在共同的方法，继承在前的会覆盖后面基类的方法
