python基础学习--类型判断及转换

发布时间：2016-12-9 标签:python

函数，是可重用的代码块，通常可以分为内置函数和自定义函数两大类，如前面接触使用到的打印函数print就是内置函数，到目前为止我们都是使用的内置函数完成一些基本常见的操作，而对于个性化的业务逻辑，就需要自定义函数，今天主要对自定义函数做一个大概的介绍。

1.定义

def hello():
   print('this is hello')
hello() # this is hello

备注：

1.def关键字+函数名([param1,param2,....])

2.参数

使用： 

def sayName1(name):
   print('my name is',name)
sayName1('neo') # my name is neo
#-----default param-----
def sayName2(name='neo'):
   print('my name is',name)
sayName2() # my name is neo
sayName2('jack') # my name is jack
#-----*args param-----
def sayName3(*args):
   print(type(args)) # <class 'tuple'>
   print('my name is',args)
sayName('jack','rose') # my name is jack rose
#-----**kwargs param-----
def sayName4(**kwargs):
   print(type(kwargs)) # <class 'dict'>
   print(kwargs)
sayName(name='jack',age=20) # {'name':'jack','age':20}

备注：

1.函数参数可以指定默认值
2.*args表示任意参数，参数在函数体内以tuple形式存在
3.**args表示任意参数，参数在函数体内以dict形式存在
4.当同时包含*args和**kwargs的时候，*args在前

3.作用域

说明：python引用变量顺序：当前作用域局部变量--外层作用域变量--当前模块的全局变量--python内置变量，与js类似。 

a='outer'
def sayName1():
   print(a)
sayName1() # 'outer'
def sayName2():
   a='inner'
   print(a)
sayName2() # 'inner'

备注：

函数内声明的变量默认情况下只在函数体内可见，即作用域位于函数内，外部的变量对于函数来说是可见的，因此对于sayName1，可以访问外部的a变量

4.global

说明：声明全局变量 

def test1():
   global num
   num=10
test1()
print(num) # 10 
def test2():
   num=10
test2()
print(num) #NameError

备注：

global声明全局变量，因此可以在其他地方访问该变量，而对于没有使用global声明的变量，在外面是不可见的，也就是超出了变量的作用域

num=10
def test3():
   num+=1
test3() # UnboundLocalError

备注：

1.在函数或者其他局部作用域中使用或者声明全局变量，如果不修改全局变量，只是简单的引用可以省略该关键字
2.在函数体内如果修改了同名的全局变量num,解释器认为是局部变量，但是局部变量num没有赋值，直接参与运算，因此报错

5.nonlocal

说明：用来在函数或者其他作用域中使用外层(非全局)变量。 

num=1 #全局变量
def test(): 
     num=2 #局部变量
     def t1():
          nonlocal num #引用局部变量
          num=3
          print(num)
     def t2():
          print(num) #引用局部变量 
     t1()
     t2()
     print(num) #引用局部变量
test()
print(num) #引用全局变量
#-----result-----
3
3
3
1

6.return

说明：表示从一个函数中返回，默认返回None,也可以返回其他对象。 

def test(a=1,b=2): 
     return a+b
print(test()) # 3
