python基础学习--基本类型

发布时间：2016-12-9 标签:python

最近在学习pyhton这门编程语言，学习一门语言，自然就是从最基础的知识点开始了，本次记录了python的基本数据类型，包含数值、字符串、列表等，与其他诸如java,javascript等编程语言而言，这些数据类型除了写法和方法使用有不同外，有很多相通之处，后面会将python与javascript语言做一个对比。

python诞生于上世纪90年代初，其作者Buido van Rossum为了打发圣诞节的无趣开发了这门语言，牛人的世界我们不懂，只能默默感叹，这让我想到了javascript的作者Brendan Eich只用了一周多的时间就开发出来了，真不知道他们咋想的，玩似的就改变了hello world的世界，牛人的境界这辈子恐怕赶不上了，那就好好学习吧。python这门语言在TIOBE编程排行榜上长期排名前五，其欢迎程度在全世界可见一斑，其信仰的编程哲学就是简单。言归正传，下面介绍一下python的基本知识。

1.int

说明：整数类型

声明： 

a=1
b,c=2,3
print(a) # 1
print(b) # 2
print(c) # 3

计算： 

a,b=2,3
print(a+b) # 5
print(a-b) # -1
print(a*b) # 6
print(a**b) # 8
print(b/a) # 1.5
print(b%a) # 1
print(b//a) # 1

备注：

1.n**m表示n的m次方
1.n//m表示除数取整

数值交换： 

a=1
b=2
a,b=b,a
print(a) # 2
print(b) # 1

2.float

说明：浮点类型

声明： 

a=1.0
b,c=2.0,3.0
print(a) # 1.0
print(b) # 2.0
print(c) # 3.0

计算： 

a,b=2.0,3.0
print(a+b) # 5.0
print(a-b) # -1.0
print(a*b) # 6.0
print(a**b) # 8.0
print(b/a) # 1.5
print(b%a) # 1.0
print(b//a) # 1.0

备注：

1.与js一样，不能精确表示和计算浮点类型，0.1+0.2永远不等于0.3

3.Decimal

说明：十进制数学计算

声明： 

from decimal import Decimal
a,b=Decimal(2),Decimal(3)
print(a) # Decimal('2')
print(b) # Decimal('3')
print(a/b) # Decimal('0.666666666666666666666667')

备注：

1.存储为十进制序列，可以设置计算进度，默认为28位，如果常规计算不能满足精度要求，可以采用decimal

4.Fraction

说明：分数

声明： 

from fractions import Fraction
Fraction(1,2) # 1/2
Fraction(1,2)+Fraction(2,3) # 1/2+2/3=7/6
Fraction(1,2)-Fraction(2,3) # 1/2-2/3=-1/6
Fraction(1,2)*Fraction(2,3) # 1/2*2/3=1/3
Fraction(1,2)/Fraction(2,3) # 1/2/(2/3)=3/4
Fraction(4,6)=>Fraction(2,3) # 自动约分
Fraction(1,2)+1 # Fraction(3/2)
Fraction(1,2)+1.0 # 1.5
f=Fraction(1,2)
f.numerator # 1
f.denominator # 2

5.string

说明：字符串

声明： 

s1='abc'
s2="abc"
s3='''abc
edf
'''

备注：

1.python没有字符类型
2.单引号或者多引号声明字符串
3.多行字符串使用'''
4.字符串不可变

使用： 

s1='abc'
#------索引------
s1[0] #'a'
s1[-1] #'c'
#------遍历------
for i in s1:
print(i) # a b c
#------切片------
s1[0:1] #'a'
s1[0:2] #'ab'
s1[:] #'abc'
s1[0:1]='123' #TypeError
#------拼接------
s1+'123' #'abc123'
s1*2 #'abcabc'
#------替换------
s.replace('a','1') #'1bc'
#------查找------
'a' in s # True 
s.index('a') # 0 
s.index('d') # ValueError:substring not found
s.count('a') # 1 
s.count('d') # 0 
s.startswith('a') # True 
s.endswith('c') # True 
#------分割------
s.split('b') #['a','c']

6.list

说明：列表

声明： 

l1=['a','b','c',1,2,3]

备注：

1.list可以存放不同类型的元素
2.有序

使用： 

#------索引------
l1[0] # 'a'
#------遍历------
for i in l1:
print(i) # a b c 1 2 3
#------切片------
l1[0:3] # ['a','b','c']
#------添加元素------
l1.append(4) #['a','b','c',1,2,3,4] 队列末尾添加
l1.insert(1,'d') #['a','d','b','c',1,2,3,4] 指定位置添加
#------获取元素------
l1.pop() # 4 默认获取队尾元素，并删除在队列中的位置
l1.pop(2) # 'b'
#------替换------
l1=['a','b','c']
l1[0:1]=[1] # [1,'b','c']
#------删除------
del l1[0:2] #['c'] 
del l1 #删除整个l1
l1.remove('a') #['b','c']
l1.remove('d') #ValueError 删除不存在的元素会报错
#------复制------
l2=l1[:] #['a','b','c']
l3=l1[0:] #['a','b','c']
l4=l1.copy() #['a','b','c'] 浅拷贝
#------合并-----
l1=['a','b','c']
l2=[1,2,3]
l1.extend(l2) #将l2的数据加入到l1里面
#------查找-----
l1=['a','b','c','a']
'a' in l1 # True
l1.index('a') # 1
l1.index('e') # ValueError 元素不存在的会报错
l1.count('a') # 2
l1.count('e') # 0

6.dict

说明：键值对集合

声明： 

d1={'name':'neo','age':30}

备注：

1.key不能重复
3.key值不可变(内容和指向)，因此不能使用list作为key
2.存储无序，查找快速

使用： 

#------访问------
d1['name'] # 'neo'
d1['address'] # KeyError
d1.get('name') #'neo'
d1.get('name') # None
#------遍历------
for k in d1:
   print(k,kv[k])
for k,v in d1.items():
   print(k,v)
#------添加------
d1['address']='wuhan'
d1['age']=30 #覆盖已存在的值
#------删除------
del kv['a'] #key值不存在会报错
del kv #删除整个变量 kv不存在会报错
#------索引------
'name' in d1 # True
'school' in d1 # False

7.set

说明：key值集合

声明： 

s1=set(['a','b','c'])

备注：

1.key不能重复
3.key值不可变(内容和指向)，因此不能使用list作为key
2.存储无序

使用： 

#------索引------
s1=set(['a','b','c'])
'a' in s1 # True 因为无序，所以不能通过位置访问
#------遍历------
for key in s1
      print(key) # c a b
#------添加------
s1.add('d') #如果元素存在，将不会添加成功
#------删除-----
s1.remove('a')
s1.remove('d') #KeyError 删除不存在的元素将会保存

8.tuple

说明：有序列表

声明： 

t1=('a','b','c')

备注：

1.不能更改元素，不能更改元素是指向不变，因此可以添加list元素
2.没有增删改

使用： 

#------访问------
t1[0] # 'a'
'a' in t1 # True
#------遍历------
for i in t1
      print(i) # a b c
